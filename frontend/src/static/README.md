# Static Assets

Questa cartella contiene le risorse statiche del frontend.

## Struttura

- `css/` - File CSS personalizzati
- `images/` - Immagini, loghi, icone
- `fonts/` - Font personalizzati

## Utilizzo

Per importare file da questa cartella:

```javascript
// CSS
import './static/css/custom.css';

// Immagini
import logo from './static/images/logo.png';
```
